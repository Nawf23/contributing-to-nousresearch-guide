## 🚀 Quick Start

**Already know what you're doing? Here's the TL;DR:**

```bash
# 1. Fork github.com/NousResearch/atropos

# 2. Clone
git clone https://github.com/YOUR-USERNAME/atropos.git
cd atropos

# 3. Setup environment
python3.11 -m venv .venv  # Mac
python3 -m venv .venv     # Windows
source .venv/bin/activate

# 4. Install dependencies (takes 15-30 mins)
pip install --upgrade pip
pip install -e ".[dev]"
pre-commit install

# 5. Create branch
git checkout -b docs/your-contribution

# 6. Make changes, commit, push
git add .
git commit -m "docs: clear description"
git push origin docs/your-contribution

# 7. Create PR on GitHub
```

**New to this?** Follow the [complete guide](CONTRIBUTING-TO-NOUS-GUIDE.md) instead!
